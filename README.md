HPIT (The Hyper-Personalized Intelligent Tutor)
===============================================

###Getting Started

1. Make sure that you have Python 2.7x and virtualenv installed.
2. Clone this repository with `git clone https://github.com/tutorgen/hpit.git`
3. Change to the project directory. `cd hpit`
4. Create a new virtual environment with `virtualenv development`
5. Activate the virtual environment with `source development/bin/activate`
6. Install the project dependencies with `pip install -r requirements.txt`
7. Run the django development server with `python manage.py runserver`
